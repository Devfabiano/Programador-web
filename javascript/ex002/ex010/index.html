Comando de repetições

while - Esta instrução é usada quando não sabemos quantas vezes quando um determinado bloco de instrução precisa ser repetido. com ele, a execução das instruções vai continuar até que uma condição seja verdadeira Cada interação

O seguite laço while intera enquanto n é menor que 3

var n = 0;

var x = 0;

while(n < 3 ){

n++;

x+ = n;

}

Cada interação incrementa n e soma a x . Portanto n e x assume os seguintes valores

No entanto, e se você quiser percorrer os carros e encontrar um especifico? e se vc não tivesse 3 corros, mas 300?

A solução é um Array!

Arrays são objetos

Arrays são um tipo especial de objetos. O tipo de operador em Javascript retorna " objeto" para Arrays.

Arrays usam numeros para acessar seus "elementos".

Neste exemplo, a pessoa[0] retorna Carlos;

var pessoa = ["carlos", "Silva", 46];

Os objetivos usam nomes para acessar seus "membros". Neste exemplo, pessoa.premeiroNome retorna Carlos;

var pessoa = {primeiroNome: "Carlos", lastName:"Silva", age 46};
